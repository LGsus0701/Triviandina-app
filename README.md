1. Descripción General del Proyecto
Triviandina es una aplicación móvil desarrollada en Android Studio con una API y una base de datos PostgreSQL. Está diseñada como una plataforma de trivia educativa sobre la cultura andina,deporte,farandula y arte permitiendo que el usuario interactúe con preguntas de opción múltiple relacionadas  la seccion de categorias .
1.1 Finalidad y Para Qué Se Usa Triviandina
La finalidad principal de Triviandina es promover el aprendizaje l de forma interactiva, dinámica y accesible desde un dispositivo móvil. La aplicación busca reforzar conocimientos mediante el juego, mejorar la memoria, la lógica y la comprensión ,accesible para todo tipo de usuario.

2. Tecnologías Utilizadas
•	Backend: Node.js, Visual Studio,POSTGRESQL
•	Frontend: Android Studio, Java/Kotlin, XML
3. Funcionalidades Principales
Aplicación Android
•	Pantalla de inicio
•	Registro y validación
•	Listado de categorías
•	Listado de preguntas
•	Registro y envío de puntajes

Conclusión
Triviandina es una aplicación educativa que combina una app Android, una API en Node.js y una base de datos PostgreSql para ofrecer trivias dinámicas sobre cultura andina y otros temas. El proyecto logró integrar todas sus partes correctamente, permitiendo registrar usuarios, mostrar preguntas y guardar puntajes. En conjunto, demuestra un sistema funcional, sencillo y útil para el aprendizaje interactivo.
